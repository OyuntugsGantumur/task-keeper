PROJECT TITLE: TaskKeeper
PURPOSE OF PROJECT: The goal of this project is to develop a task tracker that stores the information about the deadline, tag and activity of each task and keeps track of them.
VERSION or DATE: 04/26/2021
HOW TO START THIS PROJECT:

The main method is in the TaskKeeper class. The configuration file name is test.txt, so after you enter "load-tasks" or "ltk", type "test.txt". So far all commands except for search methods works properly, specifically working methods are:
	- display-tasks or dtsk
	- add-tag or atg
	- delete-tag or det
	- display-tags or dit
	- add-task or atk
	- save-tasks or stk
	- load-tasks or ltk
	- exit or e
